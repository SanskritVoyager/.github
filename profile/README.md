# Sanskrit Voyager

Welcome to **[Sanskrit Voyager](https://sanskritvoyager.com)**, a web application to explore the *Sanskrit* literature.

The application was made with the following two goals: 

1. Making the original Sanskrit texts accessible to interested readers with no knowledge of the language
2. Creating digital tools that can be used by researchers to explore the literature more easily

## Features

- **[Dictionary Search](https://sanskritvoyager.com):** Transliteration, inflection and sandhi insensitive multi dictionary search
- **[Text Analysis Tool](https://sanskritvoyager.com):** Transliterate and add interactive analysis on your text by clicking on words
- **[Book Reader](https://sanskritvoyager.com):** Explore interactively the Gretil and Sarit book libraries
- **[Corpus Search](https://sanskritvoyager.com):** Search in the corpus of the Sanskrit literature
